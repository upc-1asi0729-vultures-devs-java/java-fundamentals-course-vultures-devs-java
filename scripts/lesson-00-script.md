# Video 00 - Elevator pitch del curso

## Objetivo del video

Presentar el curso **Java desde cero: crea tu aventura de bloques** y motivar al estudiante a aprender Java mediante un proyecto práctico inspirado en juegos de bloques y exploración.

Este video funciona como introducción general del curso. No incluye código.

## Público objetivo

Estudiantes de secundaria entre 12 y 17 años que desean aprender programación desde cero usando Java.

## Idea central del curso

El curso enseñará fundamentos de Java mediante una aplicación de consola llamada **Aventura de Bloques en Java**.

El estudiante construirá paso a paso un programa donde un jugador podrá:

- Ver su estado.
- Controlar vida y energía.
- Minar bloques.
- Explorar una cueva.
- Descansar.
- Guardar objetos en un inventario.
- Usar conceptos básicos de programación orientada a objetos.

## Conceptos que se presentarán en el curso

- Java y consola.
- Variables.
- Tipos de datos.
- Operadores.
- Condicionales.
- Bucles.
- Métodos.
- Entrada de datos con `Scanner`.
- Clases y objetos.
- Constructores.
- Inventario con `ArrayList`.

## Estructura sugerida del video

### Apertura

Presentar una pregunta motivadora:

> ¿Te gustaría crear tu propia aventura de bloques usando Java?

### Problema

Explicar que muchas personas creen que Java es difícil al inicio, pero que puede aprenderse paso a paso mediante ejemplos cercanos y prácticos.

### Solución

Presentar el proyecto del curso: una aventura de bloques por consola donde el jugador podrá minar, explorar, descansar y guardar objetos.

### Proyecto final

Mostrar una idea general del menú final:

```txt
=== Aventura de Bloques ===
1. Ver estado del jugador
2. Minar bloque
3. Explorar cueva
4. Descansar
5. Ver inventario
6. Salir
```

### Llamado a la acción

Invitar al estudiante a continuar con la primera lección, donde escribirá su primer programa en Java.

## Material visual sugerido

El video puede utilizar una presentación breve con estas diapositivas:

1. ¿Te gustaría crear tu propia aventura de bloques?
2. Programar puede parecer difícil al inicio.
3. Aprenderemos Java creando un pequeño juego de consola.
4. ¿Qué aprenderás?
5. ¿Listo para empezar?

## Nota importante

No se deben usar logos, música, imágenes oficiales ni recursos protegidos de juegos comerciales. El curso debe presentarse como una aventura de bloques genérica creada con fines educativos.
